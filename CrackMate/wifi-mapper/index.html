Welcome to CrackMate

CrackMate is a university project developed to analyze the security of WiFi networks within an urban environment. The system performs real-time scanning of wireless access points, captures handshake data where possible, and uses cracking techniques to determine the vulnerability level of each network.

Detected networks are displayed on an interactive map with color-coded markers: Green for networks that were scanned (secure networks); Orange for networks where a handshake was captured but the password was not obtained; Red for networks were the handshake was captured and the password cracked. Each marker includes detailed information such as BSSID, SSID, signal strength, channel, encryption type, and GPS location.

The tool is powered by several technologies:

  • Python (subprocess): executes system-level commands (e.g., airodump-ng, aireplay-ng)
  • ADB: used to obtain GPS coordinates from a connected Android device during handshake capture
  • Tkinter: used to create the GUI of the application
  • HTML/CSS/Javascript: used for the creation of this website, the map and the charts
  • SQLite: used as database for storing and retriving data about the networks
  • JSON: used to convert the database information for the website

Captured data is stored in a CSV file (wifi_data.csv) containing fields such as BSSID, SSID, signal power, channel, encryption type, latitude and longitude. If the GPS position was obtained during the handshake capture phase, a LocationSource column is added with the value "HandshakeCapture" for that network.

During the cracking phase, the tool attempts a dictionary-based attack on each captured handshake to determine if the password can be retrieved. Results are then aggregated in the statistics page, showing encryption trends, number of networks analyzed, and crack success rate.

Disclaimer: This project does not promote illegal activity. All operations were conducted in authorized or controlled environments, and the data is used exclusively for educational and research purposes. Our goal is to raise awareness about wireless security, encourage the adoption of modern encryption standards like WPA3, and highlight the risks of using outdated protection mechanisms.
